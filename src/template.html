Projects

Real time signal processing with THz radar

Developed a real-time control software for a novel terahertz radar system using multi-processing to handle data processing and visualization without losing the measurement speed.

The software involves the control of high-frequency generator sweep, 3D geometry re-construction with time-of-flight signal and inner-frame phase-shift extraction with fast fourier transformation.

The result is submitted to IRMMW-THz international conference.

IRMMW-THz 2020 Talk

See through heart beat measurement

By taking advantage of THz wave transparency effect, skin movement by the conjunction of the heart is measured in real time with the radar while the participant has the clothes on.
